风电运检员值班指南

本文最后更新于 2025-12-07T22:43:12+08:00

鸡冠山：

  • 一期（东气）：#01-#33（共32台）
  • 二期（华锐）：#34-#66（共33台）

珠山：

  • 三期（上气）：#01-#24（共24台）
  • 四期（三一）：#25-#56（共32台）

AM 6：00——7：00 （在功率预测群中发送5张照片）

  1. 鸡冠山功率预测首页1张

相关事项：

  1. 后台数据显示——>数据生成——>生成——>扫二维码上传小程序（嘉越功率预测助手）
  2. 后台数据显示——>实时NWP查询——>记录70m风速（填写鸡冠山运行日志并在早班会进行汇报）
  1. 珠山功率预测首页1张

相关事项：

  1. 后台数据显示——>数据生成——>生成——>扫二维码上传小程序（嘉越功率预测助手）

  1. 兆方美迪首页1张

  2. 两细三率2张（鸡冠山、珠山各1张）

    360浏览器——>网厂交互平台——>新能源并网运行考核——>基础日数据——>修改日期（数据不合格在早班会进行汇报）

A.M. 7：30 （早班会进行汇报）

  1. 一、二次设备运行正常（看综自电脑）
  2. 今日风速预测：
  3. 夜间故障情况：
  4. 两细三率数据不合格情况：

A.M. 8：00——

  • 电脑上：

    1. 鸡冠山日报、珠山日报

    2. 日工作汇报表、佳木斯片区日报表

    3. 并网运行考核数据（早晚班结束立刻填写）

      在日工作汇报表的sheet中寻找

    4. 现货月各厂站0-24时发电量统计（写昨日数据、10点前完成）

    5. 智慧平台

      1. 鸡冠山、珠山专项任务班前会

      2. 鸡冠山、珠山全自动巡检

      3. 鸡冠山、珠山图像告警管理

      4. 鸡冠山、珠山定期工作任务管理站内巡视

      5. 工作票流程

  • 纸质本：

    1. 鸡冠山、珠山工作日志
    2. 鸡冠山、珠山风机工作票工作簿
    3. 桌面巡检告警信息记录
    4. 三期、四期视频监控巡视记录

P.M. 15：00（上传珠山发电量、最大有功功率情况Word到QQ群–珠山15时汇报群）

word1:外网电脑桌面——>珠山一期情况汇报

​ 上气SCADA电脑——>表格报表——>报表统计——>统计数据——>勾选发电量、最大有功功率——>时间当天、最大值——>导出

word2:外网电脑桌面——>珠山二期情况汇报

​ 三一SCADA电脑——>报表——>自由报表——>风机全选、日期当天——>勾选发电量、最大有功功率——>导出_

P.M. 16：00（首发故障导出，打印前删除当天以外的数据）

  1. 一期东气SCADA电脑：

    日志管理——>风机故障日志——>选择设备（全选）——>只要首发故障、日期当天——>查询——>导出并打印签字

  2. 二期华锐SCADA电脑：

    log日志——>报表——>保存——>打印签字（删除当天以外的数据）

  3. 三期上气SCADA电脑：

    数据查询——>首发故障——>日期当天、风机全选、首发故障——>查询——>导出并打印签字

  4. 四期三一SCADA电脑：

    报表——>故障统计——>日期当天、风机全选、首发故障——>查询——>导出并打印签字

P.M. 19：00（报哈调，电话：51235555）

继保室后面抄表——>总上网 = 达风线（去掉小数点）——>填写报哈调记录表

P.M. 19：30（晚班会进行汇报）

  1. 一、二次设备运行正常
  2. 风机工作票情况（哪台风机干了什么，工作票已押回、收回、未收回）
  3. 生产群和公司群里面的通知
  4. 限电情况
  5. 复位不了的风机出现的故障
  6. 巡检机器人正常：统计分析——>危急告警、严重告警、一般告警（查看指示灯是否误报）

P.M. 20：00——21：00（智慧平台交接班，按月填写）

P.M. 21：00——24：00（完成2个日报、2个网厂、故障风机损失电量表、风机缺陷台账）
