【引入方式】

JavaScript 程序不能独立运行，它需要被嵌入 HTML 中，然后浏览器才能执行 JavaScript 代码。通过 script 标签将 JavaScript 代码引入到 HTML 中，有三种方式：

内部JavaScript

通过 script 标签包裹 JavaScript 代码，直接写在html文件里

书写的位置尽量写到文档末尾前面

外部JavaScript

一般将 JavaScript 代码写在独立的以.js结尾的文件中，然后通过 script 标签的 src 属性引入

如果 script 标签使用 src 属性引入了某 .js 文件，那么标签的代码会被忽略！！！
